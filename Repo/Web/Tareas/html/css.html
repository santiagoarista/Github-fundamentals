¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para definir el estilo y apariencia de documentos HTML.

Conceptos Básicos de CSS

¿Qué son los selectores?

Los selectores en CSS se utilizan para apuntar a los elementos HTML y aplicarles estilos.

Propiedades y Valores

Las propiedades definen qué aspecto del elemento se va a modificar, y los valores determinan cómo será esa modificación.

Ejemplo:

p { color: blue; font-size: 16px; }

Tipos de Selectores

  • Selector de Tipo: Se refiere a los elementos HTML directamente. Ejemplo: p { color: red; }
  • Selector de Clase: Se define con un punto (.) y se aplica a múltiples elementos. Ejemplo: .destacado { font-weight: bold; }
  • Selector de Identificador: Se define con un numeral (#) y es único para un elemento. Ejemplo: #titulo { text-align: center; }
