Comandos de Terminal

En este bloque hemos utilizado distintos comandos para utilizar Git y Github desde nuestra terminal

Tabla de comandos

Comando Descripción
cd Sirve para moverte dentro de tus carpetas
ls Sirve para ver el contenido de la carpeta en la que estás
mkdir Crear una carpeta
touch Crear un nuevo documento
git add Darle seguimiento a cierto documento
git commit Guardar cambios en tu carpeta local
git push Guardar cambios en tu carpeta en la nube
